# CHIP-8 Emulator

This program emulates the fictional CHIP-8 computer. Not all instructions are defined yet, and most programs will fail to run properly. This is still a work-in-progress.

## Installation

Compile with `gcc`:

```bash
make
```

## Usage

```bash
./chip8 <path-to-rom>
```
